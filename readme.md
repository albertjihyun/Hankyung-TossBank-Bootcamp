**Duration:** Feb 13, 2026 – Aug 13, 2026  
**Track:** Full-Stack Development  
**Focus:** System Thinking, Real-World Financial Services, Product-Oriented Development

## 📌 Overview

This repository documents my journey through the Toss Full-Stack Bootcamp.  
The program emphasizes not only implementation skills, but also **system design, logical reasoning, and real-world product thinking**.

Throughout the bootcamp, I build and iterate on projects in the **financial domain**, leveraging APIs and designing user-centered services.

---

## 🧩 Program Structure

### 🏃 Weekly Learning & Practice
- Frontend & Backend fundamentals
- Database design and querying
- Full-stack integration (Client ↔ Server ↔ Data)
- Emphasis on **architecture and execution flow**, not just coding

---

### ⚡ Hackathons (×6)
Six hackathons are held during the program to simulate real-world development cycles.

Each hackathon focuses on:
- Rapid problem solving
- Team collaboration
- MVP delivery under constraints
- Practical application of learned concepts
